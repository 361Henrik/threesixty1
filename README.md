# threesixty1 site

Independent Vite/React site for threesixty1.com.

## Scripts

- `npm run dev` starts the local development server.
- `npm run build` creates a production build.
- `npm run preview` serves the production build locally.

## Prototype

The current threesixty1.com visual direction prototype lives in `prototypes/threesixty1-visual-directions.html`.
